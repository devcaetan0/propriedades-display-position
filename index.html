Position Static

O static obedece o fluxo padrão do documento HTML, ignorando padding e margin e mantendo o elemento o máximo na extremidade possível. Não é utilizado de forma prática, servindo mais para zerar a posição do elemento desejado

Position Relative

A propriedade do relative realiza uma sobreposição sobre os elementos da página, porém o espaço que ela ocupava permanece, não sendo substituído por outros elementos. Ela é útil para criar elementos para um ponto de referência, podendo modificá-los para criar uma "âncora".

Position Sticky

Essa propriedade é um híbrido entre o fixed e o relative, onde o elemento age como relative até que a rolagem do site o faça atingir o limite da página. Após atingir alguma extremidade do documento, o elemento sticky passa a agir igual um elemento fixed.

Position Absolute

O absolute é a propriedade de maior potência na hierarquia, desrespeitando totalmente o fluxo normal do documento. Ela é o problema da divisão, um rei indomável que não altera seu lugar nem tamanho na página.
