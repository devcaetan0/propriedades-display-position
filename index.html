Display Inline Flex

O display inline flex é parecido com o flex mas ele tem um comportamento de forma inline. Ele ocupa apenas o tamanho do conteúdo ou seja fica na mesma linha que os outros elementos. Já o flex ocupa a linha inteira, quebra linha automaticamente. Flex bloco grande, inline flex pequeno e do tamanho colocado.

Display Grid

O display grid é baseado em sistema de layout em grade. Resolve problemas de layout. Diferente do flexbox, que organiza em uma unica direção o grid organiza em 2 dimensões ao mesmo tempo.
